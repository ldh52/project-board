# 게시판 서비스
패스트캠퍼스 강의를 기초로 직접 만드는 게시판 서비스입니다. 
# 개발 환경
- Intellij IDEA Ultimate
- Java 17
- Gradle 7.4.1
- Spring Boot 2.7.0
# 기술 세부 스텍
Spring Boot
- Spring Boot Actuator
- Spring Web
- Spring Data JPA
- Rest Repositories
- Rest Repositories HAL Explorer
- Thymeleaf
- Spring Security
- H2 Database
- MySQL Driver
- Lombok
- Spring Boot DevTools
- Spring Configuration Processor

그 외

* QueryDSL 5.0.0
